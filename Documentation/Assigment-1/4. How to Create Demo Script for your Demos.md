# How to create a **Demo Script** for your **Demos**?

### - GitHub & Create New Repository, Add Collaborater & Create New Branches
### - Take Screen Shots of all your Work

## ➤ First of all Create a folder on your Desktop, name it as **Demo**

![Step1](https://github.com/rahiltariq07/Demo/blob/main/Documentation/Images/4/Step1.png)

## ➤ Create a folder in Demo folder and name it as **Documentation**

![Step2](https://github.com/rahiltariq07/Demo/blob/main/Documentation/Images/4/Step2.png)

## ➤ Create a folder namely **Images** in your previous documentation folder

![Step3](https://github.com/rahiltariq07/Demo/blob/main/Documentation/Images/4/Step3.png)

## ➤ Create 3 folders in image folder for demos

![Step4](https://github.com/rahiltariq07/Demo/blob/main/Documentation/Images/4/Step4.png)

## ➤ Paste all images in it that you screen shoted first

![Step5](https://github.com/rahiltariq07/Demo/blob/main/Documentation/Images/4/Step5.png)

![Step6](https://github.com/rahiltariq07/Demo/blob/main/Documentation/Images/4/Step6.png)

## ➤ Click on **Upload file** option

![Step7](https://github.com/rahiltariq07/Demo/blob/main/Documentation/Images/4/Step7.png)

## ➤ Now drag the **Documentation** folder in it

![Step8](https://github.com/rahiltariq07/Demo/blob/main/Documentation/Images/4/Step8.png)

## ➤ Scroll down and click on **Commit Changes**

![Step9](https://github.com/rahiltariq07/Demo/blob/main/Documentation/Images/4/Step9.png)

## ➤ After commit changes it will take a little time to process

![Step10](https://github.com/rahiltariq07/Demo/blob/main/Documentation/Images/4/Step10.png)

## ➤ Congratulations your Demo Repositories has been Created Sucessfully

![Step11](https://github.com/rahiltariq07/Demo/blob/main/Documentation/Images/4/Step11.png)

## ➤ Now Click on **Code** and Click on **Code in GitHub**

![Step12](https://github.com/rahiltariq07/Demo/blob/main/Documentation/Images/4/Step12.png)

## ➤ Go to Repostories in GitHub Desktop and Click on **Show in Explorer**

![Step13](https://github.com/rahiltariq07/Demo/blob/main/Documentation/Images/4/Step13.png)

![Step14](https://github.com/rahiltariq07/Demo/blob/main/Documentation/Images/4/Step14.png)

![Step15](https://github.com/rahiltariq07/Demo/blob/main/Documentation/Images/4/Step15.png)

## ➤ Create new file in VSC
